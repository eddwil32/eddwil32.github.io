Active Directory → Recorded Future Sync

Python automation script that collects all user accounts from Active Directory via LDAP, classifies them as Active or Disabled based on the userAccountControl bitmask, and performs a delta sync to maintain two dedicated lists in Recorded Future — enabling real-time identity context for SOC alert enrichment in Splunk SIEM and SOAR playbooks.

Script Python che raccoglie tutti gli account utente da Active Directory via LDAP, li classifica come Attivi o Disabilitati tramite il bitmask userAccountControl, e sincronizza due liste dedicate su Recorded Future — fornendo contesto identità in tempo reale per l'enrichment degli alert in Splunk SIEM e nei playbook SOAR.
